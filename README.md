# Phishing Awareness Training

## CodeAlpha Cyber Security Internship – Task 2

### Project Overview

This project was completed as part of the CodeAlpha Cyber Security Internship Program.

The purpose of this project is to educate users about phishing attacks, social engineering techniques, fake websites, and cybersecurity best practices.

---

## Presentation Topics

- Introduction to Phishing
- How Phishing Works
- Types of Phishing
- Fake Websites
- Social Engineering
- Prevention Tips
- Interactive Quiz
- Conclusion

---

## Learning Outcomes

- Understanding phishing attacks
- Identifying suspicious emails and websites
- Learning social engineering techniques
- Following cybersecurity best practices
- Improving online safety awareness
